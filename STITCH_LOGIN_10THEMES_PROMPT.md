THEME 1: "DARK LUXURY GOLD" (Premium Confidence)
────────────────────────────────

Target Audience: High-end training centers, elite volleyball programs
Vibe: Sophisticated, trustworthy, premium, "pure gold"

Color Palette:
├─ Primary: #EAB308 (Gold) — CTA, highlights, active states
├─ Dark Background: #0c0c18 (Ultra Dark Navy)
├─ Secondary BG: #14141f (Slightly lighter navy for cards)
├─ Text Primary: #ffffff (White)
├─ Text Secondary: rgba(255,255,255,0.6) (Muted white)
├─ Text Tertiary: rgba(255,255,255,0.3) (Very subtle)
├─ Accent: #F97316 (Orange for errors/warnings)
└─ Success: #22C55E (Green)

Typography:
├─ Headings: Lexend Bold 700 (clean, modern, geometric)
├─ Body: Lexend Regular 400–500 (readable, elegant)
└─ Labels/Small: Space Grotesk 600 (technical, precise)

Visual Style:
├─ Glassmorphism hero (blur 20px background)
├─ Subtle shadows (not harsh, luxury feel)
├─ Border radius: 12px (buttons), 16px (cards), 18px (nav)
├─ SVG court: Stroke reveal animation (1.2s, cubic-bezier)
├─ Card states: Soft gold border on active, subtle glow
└─ Overall Aesthetic: Dark mode purist, gold as single accent point

Key Visual Elements:
├─ Hero: SVG court (quadra) with smooth stroke animation, shimmer gold text overlay
├─ Role Cards: Glassmorphic, blur 20px, gold border on active state
├─ Buttons: Gold gradient (135deg), black text, scale-tap feedback (0.97)
├─ Inputs: Dark background (#1a1a1a), gold border on focus, subtle glow
└─ Animations: Shimmer text (gold wave), glow pulse (soft), fade-up entrance

Suggested Micro-Interactions:
├─ Input focus: Gold glow + subtle shadow expansion (200ms)
├─ Button hover: Opacity 0.9, slight lift (translateY -2px)
├─ Role card click: Scale 1.02 → 1.0, border color to gold (150ms)
├─ SVG stroke: 1.2s reveal on page load, cubic-bezier smooth
├─ Error state: Red border + shake animation (shake 3 cycles, -8px)
└─ Loading: Gold spinner or dotted pulse (120 bpm rhythm)

# WILLPRO — Master Memory

Memória viva do projeto **Will Treinos PRO**: interações, decisões e alterações relevantes feitas no Cursor (ou manualmente), **sempre com data e hora**.

## Onde registrar

- Arquivo principal: **`REGISTRO_INTERACOES.md`**
- Formato: entradas em **ordem decrescente** (a mais recente no topo).
- Cada entrada deve incluir **data/hora com fuso** (ex.: `-03:00` America/São_Paulo) e bullets objetivos (o quê / arquivos / motivo).

## Para assistentes (IA)

Ao encerrar uma sessão com mudanças de código ou decisões de produto: **acrescente uma nova seção datada** no topo de `REGISTRO_INTERACOES.md`. Não substituir o histórico; apenas **anexar no topo**.

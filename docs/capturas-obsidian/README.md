# Capturas Obsidian / sessão


Imagens guardadas aqui com nome **`obsidian-workspace_YYYY-MM-DD_HH-mm-ss.png`** (data e hora locais no momento da cópia).

Quando pedires para **adicionar esta imagem** (captura do Obsidian ou equivalente), o assistente deve:

1. Colocar o ficheiro em **`docs/capturas-obsidian/`** (criar a pasta se não existir).
2. Usar o prefixo acima + timestamp no nome do ficheiro.

Para usar dentro de um **Cofre Obsidian** noutro disco, copia ou liga esta pasta ao vault (por exemplo pasta anexada ou sync).

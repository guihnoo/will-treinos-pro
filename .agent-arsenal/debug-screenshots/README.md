# Prints de debug (Claude Code / Cursor)

Use esta pasta para **enviar capturas de tela** aos agentes sem poluir o Git.

## Fluxo recomendado

1. Salve o print aqui com nome curto, por exemplo: `bug-notificacao-data.png`.
2. No **Claude Code** (terminal), referencie o ficheiro:
   - Arraste o ficheiro para o terminal, ou
   - Escreva o caminho completo, por exemplo:
     `C:\Users\monte\Desktop\will-treinos-pro\.agent-arsenal\debug-screenshots\bug-notificacao-data.png`
3. No **Cursor**, pode usar `@` e escolher o ficheiro de imagem na árvore do projeto.
4. Depois da análise, **apague** os PNG/JPG desta pasta para libertar espaço e evitar confusão nas próximas análises.

## Git

Imagens aqui são ignoradas pelo `.gitignore` (só este `README.md` entra no repositório).

## Dica

Para o mesmo erro em várias telas, prefira **um zip** ou **vários ficheiros numerados** (`01-login.png`, `02-modal.png`) para manter a ordem da reprodução.

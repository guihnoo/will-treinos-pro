# Apresentação comercial — Will Treinos PRO

Arquivos prontos para enviar ou apresentar ao cliente.

## Download recomendado (PDF)

**Arquivo:** [`WILL-TREINOS-PRO-APRESENTACAO-CLIENTE.pdf`](./WILL-TREINOS-PRO-APRESENTACAO-CLIENTE.pdf)

- ~10 páginas, identidade dark + gold
- Conteúdo: problema, solução, 4 pilares, cockpit admin, coach, aluno, automações/IA, tecnologia e CTA
- Abra no Adobe Reader, navegador ou envie por e-mail/WhatsApp

## Versão interativa (HTML)

**Arquivo:** [`WILL-TREINOS-PRO-APRESENTACAO-CLIENTE.html`](./WILL-TREINOS-PRO-APRESENTACAO-CLIENTE.html)

- Duplo clique abre no navegador
- Botão **Salvar como PDF** na barra inferior (ou `Ctrl+P` → Imprimir em PDF)

## Regenerar o PDF (opcional)

```powershell
cd c:\Users\monte\Desktop\will-treinos-pro
node scripts/export-apresentacao-pdf.mjs
```

Requer Chromium do Playwright (`pnpm exec playwright install chromium`).
